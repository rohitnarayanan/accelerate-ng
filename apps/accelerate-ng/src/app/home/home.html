Accelerate NG Core

A foundational library providing core functionalities and utilities for building Angular applications efficiently.

Features:
  • Utility functions for common tasks
  • State management tools
  • Enhanced routing capabilities
  • Performance optimizations
ng add @rohitnarayanan/accelerate-ng-bootstrap
Usage: Import the required modules and use the provided utilities in your Angular application.

Accelerate NG Bootstrap

Custom widgets built on Bootstrap 5 themes for Angular ecosystem.

Features:
  • Native Angular components
  • Widgets like carousel, modal, popover, tooltip, datepicker, and more
  • Accessible with proper HTML elements and ARIA attributes
  • High-quality code with almost 100% test coverage
Dependencies:
  • Bootstrap
  • ng-bootstrap
  • bootstrap-table
  • jQuery TreeGrid
  • ngx-quill
Installation: ng add @ng-bootstrap/ng-bootstrap
